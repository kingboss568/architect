SECTION 01 / SCOPE

不是更多素材，是現場用得上的詳圖

營造端、技師端、設計端各自關心的「能不能落地」不同。本圖庫從交圖到簽證、從放樣到收尾，把每一個會被監造盯到的節點先做好。

01 / PRECISION

1:1 實尺寸繪製

所有節點以 1:1 實尺寸繪製，方便依比例放樣、套用至 1:20 / 1:10 / 1:5 詳圖，不需重新換算。

02 / LAYER

圖層命名統一

採用 A-/S-/M- 分組命名邏輯，便於凍結、打印或匯出至 BIM 工序，與既有事務所標準對得上。

03 / SEQUENCE

依工序整理

從放樣、結構、外牆、防水到收邊，依施工順序排列，新人也能照圖順序逐張對位。

04 / LICENSE

永久商用授權

一次買斷，可用於事務所商業專案、提案、考照練習與教學參考，下載連結 7 天內可重新下載。

SECTION 02 / DRAWING SCHEDULE

圖面清冊：節點、剖面、大樣、收邊一次整理好

完整包不是把圖塊堆在一起，而是依「結構 → 外殼 → 內裝 → 收邊」的工序排列，搭配尺寸標註、圖層命名與比例樣板。

DT-01結構詳圖

RC、SRC、SS 柱梁節點、配筋大樣、樓板開口補強，含尺寸、鋼筋編號與層次。

DT-02牆身剖面

外牆、隔間、防火牆斷面剖面，含飾面、保溫層、防水層、底層的層厚與材料。

DT-03樓梯大樣

RC 樓梯、鋼樓梯、扶手與級高級深，含淨寬、起步距與級頂收邊。

DT-04門窗收邊

門窗框、窗台、過樑、五金、玻璃膠條與導水條的標準收邊大樣。

DT-05防水節點

屋頂女兒牆、外牆防水、衛浴止水墩、外露管線等高風險節點。

DT-06收邊大樣

天花、地坪、踢腳、轉角、燈槽、伸縮縫等室內外收邊處理。
